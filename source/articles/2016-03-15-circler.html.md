### コマンド一覧 (READMEより抜粋)

#### watch
リアルタイムにビルドのログを監視します

```sh
$ circle watch
```

#### projects
リポジトリの一覧を表示します

```sh
$ circle projects
```

#### builds
ビルドの一覧を表示します

```sh
$ circle builds
```

#### build
ビルドの詳細を表示します

```sh
$ circle build
```

#### browse
CircleCIの該当ページをブラウザで表示します

```sh
$ circle browse
```
